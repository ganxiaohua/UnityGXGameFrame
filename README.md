# UnityGXGameFrame
* 需要odin插件 yooassets插件 unitask插件 fgui插件
* 测试案例：https://github.com/ganxiaohua/GXGame
* 详情知乎：https://zhuanlan.zhihu.com/p/2022215951286899930

集成UI框架(基于fgui),资源加载框架(基于yooAssets)

1.节点分布,整个游戏的功能都能在编辑器上找到相对应的节点
2.强大的UI系统,适配市面上绝大多数游戏的需求.
3.数据驱动,内存严格排列,数据紧密,调用和查找都无比快速
4.逻辑和渲染的分离,即便脱离unity也能使用.
5.逻辑结构清晰,根据框架写的代码非常容易维护.
6.强大的编辑器工具,无论是UI还是运行时候的逻辑和数据都能快速查看和修改方便定位问题.
